What is BSF?

The Bean Scripting Framework (BSF) is an architecture for incorporating scripting into Java applications and applets.

It is used for example in the Xalan XSLT processor and in the Jakarta BSF Taglib.

With JRuby you can also use Ruby as a scripting language in this applications.

How to use JRuby with BSF?

  1. First you need to download BSF.
  2. JRuby supports BSF but BSF doesn't support JRuby. There are two possibilities:
    1. Add this line to the com/ibm/bsf/Language.properties file: 
      jruby = org.jruby.javasupport.bsf.JRubyEngine, rb
    2. Download a patched version of the bsf.jar file
  3. Now you can use JRuby like the other scripting languages in BSF.

    For example: 

    BSFManager bsfManager = new BSFManager();
bsfManager.eval("jruby", "script", 0, 0, "puts \"Hello from JRuby\"\n");
