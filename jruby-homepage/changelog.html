Changes in beta 0.2.4

  • Add support for Proc class
  • Add support for Method class
  • Add support for Bignum class
  • Add support for JavaInterface class
  • Add support for Math module
  • Improve support for Module, Array and Hash class
  • Add support for Unit test with JUnit
  • Fix a lot of bugs

Changes in beta 0.2.3
  • Add Objectspace module
  • Add Method class
  • Add some methods to Module class
  • Add several methods to the Array class.
  • Fix a lot of bugs (blocks, Java support).

Changes in beta 0.2.2
  • Add some methods to the Array class.
  • Complete support for exceptions.
  • Add swing example.
  • Fix some bugs.

Changes in beta 0.2.1
  • Add a lot of methods to the Array class.
  • Add support for BSF.
  • Add support for exceptions.
  • Rewrite the interpreter.
  • Improve performance.
  • Fix a lot of bugs.
  • Update build.xml, now extract tar.gz in subdirs

Changes in beta 0.2.0
  • Add support for loading Ruby files (Kernel#require).
  • Add support for Java 1.4 regular expressions.
  • Add pluggable regular expressions engines support.
  • Add some methods to Kernel class (sprintf, prinf, format)
  • Add some methods to String class (index, rindex, succ, next, upto)
  • Refactorize ID / RubyId and Ruby.

Changes in alpha 0.1.9
  • Add support for loading JAR libraries (Kernel#require).
  • Add support for substitution of Strings (String#sub, String#gsub, ...).
  • Add support for evaluation of Strings ("#{ expr }").
  • Add support for here documents (<<END_OF_STRING ... END_OF_STRING).
  • Add RegExp and MatchData classes.
  • Refactorized Parser and Scanner (updated to Ruby 1.6.5).
  • Add support for commandline args (ARGV).
  • Add a new stylesheet for the xml example.

Changes in alpha 0.1.8
  • Add fixnum-cache.
  • Add method-cache.
  • Add all methods to RubyRange.
  • Java scripting support improved:
    • Automatic load Java classes when they are returned by a method.
    • Support for Object and Arrays
  • Add "-e" switch to Main class.
  • Fix a lot of bugs in the runtime, interpreter and parser.
  • Add fib.rb sample.
  • Fix some bugs in the samples.

Changes in alpha 0.1.7
  • Fix some bugs in the parser.
  • Fix some bugs in the src tarball jruby-src-0_1_6b.tar.gz
  • Fix some bugs in the Java support.
  • Update some classes to Ruby 1.6.5.
  • Add an other example for Java support (samples/xml.rb)
  • Add some methods to Array.
  • Add some methods to String.
  • Add some methods to Module.
  • Add module Enumerable.
  • Add some methods to Enumerable.

Changes in alpha 0.1.6b

  • Update parser to Ruby 1.6.5
  • Fix broken tarball jruby-src-0_1_6.tar.gz
  • Fix a lot of bugs in the Java support.
  • Add an example for Java support (samples/java2.rb)

Changes in alpha 0.1.6

  • Add support for scripting Java classes. (Add class JavaObject).
  • Add class Array.
  • Add some methods to Array.
  • Add some methods to Class.
  • Add class Symbol.
  • Add some methods to Symbol.
  • Add a lot of methods to Module.
  • Fix a lot of bugs in the interpreter.
  • Fix a lot of bugs in all Ruby classes.
