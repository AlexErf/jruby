A first Swing example

JavaObject.load_class "javax.swing.JFrame"
JavaObject.load_class "javax.swing.JLabel"

frame = JFrame.new("HelloWorldSwing")
label = JLabel.new("Hello World")

frame.getContentPane().add(label)

frame.setDefaultCloseOperation(JFrame::EXIT_ON_CLOSE)
frame.pack();
frame.setVisible(true);

JavaObject.load_class

The load_class method takes upto two parameters.

  1. The Java name of the class
  2. Optional. The Ruby name of the class (Default: Java name without packages)
It defines a Ruby class which maps all methods and constants to a Java class.

The arguments are automatically converted between Ruby and Java types.

If a method returns a new Java class it will be automatically be loaded.

Kernel.require (*.jar)

The Kernel.require takes one argument.

The method adds the Jar library which is specified in the first argument to the Classpath.
